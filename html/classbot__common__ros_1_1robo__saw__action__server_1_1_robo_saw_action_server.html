◆ get_no_collision_angle()

def bot_common_ros.robo_saw_action_server.RoboSawActionServer.get_no_collision_angle (   self,
  initial_pose = None 
) 
Find an angle to rotate TCP by in the x-tool direction such that it won't collide with local topology
Uses collision checker to align guard

convention of angles is w.r.t. tool-x-axis. Meaning that cutting the ceiling requires a positive rotation angle and cutting the floor requires a negative rotation angle.

steps:
* get boolean collision values for tcp +- 90 deg rotation
* get min-max valid rotations inside region where nominal_orient is
* see if nominal_orient +- margin is non-colliding
* else, find min-rotation for no-collision


returns: pose_out (type m3d.Orientation), clipped (type float32), collision (bool)
