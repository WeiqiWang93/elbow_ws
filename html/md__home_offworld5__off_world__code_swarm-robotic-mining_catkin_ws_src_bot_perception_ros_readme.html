To install astra mini - s on ros-indigo. Must install astra_camera and astra_launch from github

sudo apt-get install libudev-dev
git clone git@github.com:orbbec/ros_astra_camera.git
git clone git@github.com:orbbec/ros_astra_launch.git
catkin_make
source devel/setup.bash
./ros_astra_camera/scripts/create_udev_rules
roslaunch astra.launch

http://wiki.ros.org/rqt_plot

rosdep install rqt_plot
sudo apt-get install ros-kinetic-rqt-plot

Confirm server is running by typing

rosrun dynamic_reconfigure dynparam get [node-name]
rqt_plot /robosaw/plunge/error /robosaw/cut/error
