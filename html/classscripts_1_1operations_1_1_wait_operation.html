Detailed Description

Waits for a delay and/or for a set of operations to complete
    
        Several cases:
        
        If one of the operations the handler is waiting for fails, then
        the wait operation also fails.
        
        If the given delay terminates while the handler is waiting 
        for some operations that are not complete, the wait operation also fails.
        
        When no operation is given and the wait_ops flag is set, the wait handler
        waits for all the operations before this operation to complete.
