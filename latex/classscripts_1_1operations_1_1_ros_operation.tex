\doxysubsection{Detailed Description}
\begin{DoxyVerb}Models and handles a ROS operation

   ROS operations are ROS actions (actionlib) with the following
   life-cycle:
   - Send a goal - includes ActionType common type request from ops definition arguments 
   - Handle feedback: common progress status (ProgressStatus msg) + specific status
   - Handle result - common result handling, specific result contents are handled by telemetry sampler
\end{DoxyVerb}
