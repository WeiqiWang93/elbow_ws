The arguments associated with an operation are specified in the {\ttfamily Arguments} tab\hypertarget{md__home_offworld5__off_world__code_swarm-robotic-mining_catkin_ws_src_bot_overseer_readme_autotoc_md21}{}\doxysubsubsection{Asynchronous operations}\label{md__home_offworld5__off_world__code_swarm-robotic-mining_catkin_ws_src_bot_overseer_readme_autotoc_md21}
Several asynchronous operations that take time to complete can run in parallel. For this purpose set the flag wait\+\_\+ok to False\+:

{\ttfamily op(\char`\"{}\+L\+O\+N\+G\+\_\+\+O\+P\char`\"{}, wait\+\_\+ok=False)}\hypertarget{md__home_offworld5__off_world__code_swarm-robotic-mining_catkin_ws_src_bot_overseer_readme_autotoc_md22}{}\doxysubsubsection{Waiting}\label{md__home_offworld5__off_world__code_swarm-robotic-mining_catkin_ws_src_bot_overseer_readme_autotoc_md22}
A mission can wait for a delay or some previous operation(s) to complete. The {\ttfamily W\+A\+I\+T\+\_\+\+M\+I\+S\+S\+I\+ON} operation provides the following arguments\+:
\begin{DoxyItemize}
\item delay\+: a delay in seconds to wait for.
\item wait\+\_\+ops\+: a flag that indicates whether to wait for previous operations to complete.
\item ops\+: specifies the list of operation indexes to wait for. When not given, the wait operation waits for all the previous operations to be completed.
\end{DoxyItemize}

{\ttfamily op(\char`\"{}\+W\+A\+I\+T\+\_\+\+M\+I\+S\+S\+I\+O\+N\char`\"{}, wait\+\_\+ops=True, ops=\mbox{[}7\mbox{]})}\hypertarget{md__home_offworld5__off_world__code_swarm-robotic-mining_catkin_ws_src_bot_overseer_readme_autotoc_md23}{}\doxysubsubsection{Scripting}\label{md__home_offworld5__off_world__code_swarm-robotic-mining_catkin_ws_src_bot_overseer_readme_autotoc_md23}
A mission can also include one or several python scripts either\+:
\begin{DoxyItemize}
\item At the main scope of the mission\+: the script is executed upon {\ttfamily loading} the mission
\item Within a specific {\ttfamily S\+C\+R\+I\+P\+T\+\_\+\+M\+I\+S\+S\+I\+ON} operation scope\+: the script is executed upon execution of the operation. In particular this allows to update the mission automatically based on results from previous operations of the same mission.
\end{DoxyItemize}
